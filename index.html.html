La technologie web et les differents langage de programation

DEFINITION DE LA TECHNOLOGIE WEB

La technologie web et tout les outils et pratique adoptée pour créer des sites internet et des applications web visible via un navigateur par le biais d'un héberger comme GitHub.

Quelques Exemples de technologies Web les plus utilisé.

  • HTML : structure la page
  • CSS : donne le style (couleurs, taille, design)
  • JavaScript : rend la page interactive

      • LE LANGAGE DE PROGRAMMATION WEB

    DEFINITION

    Un langage de programmation web sert à créer des sites ou applications qui fonctionnent sur Internet. Pour le Front-end (ce que l’utilisateur voit).

    Quelques exemples de langage de programmation web.

  • HTML
  • CSS
  • JavaScript

      • LE LANGAGE DE PROGRAMMATION MOBILE

    DEFINITION

    Le langage de programmation Mobile sert à créer des applications pour téléphone Android ou iPhone.

    Quelques exemples de langage de programmation web.

  • Pour Android : Java
  • Pour iPhone : Swift
